C-compiler Implementation

 

Introduction

This is my final project for the compiler course. The compiler is written in C and uses FLEX for lexical analysis to tokenize the input code. Then, Bison is used for syntax analysis to generate an Abstract Syntax Tree (AST). Afterward, the AST is traversed to produce x86 assembly language compatible with NASM, which is then assembled using NASM to convert the assembly code into machine code. Finally, the program is executed in the DOSBox emulator.
